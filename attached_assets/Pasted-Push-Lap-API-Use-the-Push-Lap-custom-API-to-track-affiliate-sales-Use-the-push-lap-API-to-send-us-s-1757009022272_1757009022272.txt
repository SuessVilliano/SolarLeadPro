Push Lap API
Use the Push Lap custom API to track affiliate sales

Use the push lap API to send us sign up and sale events directly from your server

Step 1: Tracking clicks
Insert this line of code in your website's <head> tag to start tracking clicks:

Normal Method
Using Google Tag Manager
<script 
  src="https://pushlapgrowth.com/affiliate-tracker.js" 
  data-affiliate data-program-id="9acd7ded-50dc-4b98-b698-c78edc0481b3" 
  async>
</script>

Step 2: Create your API key
Go, to your dashboard -> API Keys -> Create a new API key. Click here for the full guide

Step 3: Tracking Sign ups
Send sign up data over to push lap using the following code:

const body = {
  affiliateId: window.affiliateId, // This could be the affiliateId or Email of the affiliate who referred the user
  name: "John Smith",
  email: "jsmith@example.com",
  referredUserExternalId: "<string>", // Optional field to keep an external userId, for example their id on your database or their stripe customer Id
  plan: "<string>", // Optional field to keep the plan name
  status: "<string>", // Optional field to keep the status of the referred user
};

const options = {
  method: 'POST',
  headers: {
    Authorization: 'Bearer <YOUR_API_KEY>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify(body)
};

fetch('https://www.pushlapgrowth.com/api/v1/referrals', options)
  .then(response => response.json())
  .then(response => console.log(response))
  .catch(err => console.log(err));

Step 4: Tracking Sales
Call the following API endpoint when a sale is made:

const body = {
  referralId: "3c90c3cc-0d44-4b50-8888-8dd25736052a", // The referral Id or email of the user who bought an item
  externalId: "<string>", // Optional field to keep an external userId, for example their id on your database or their stripe customer Id
  externalInvoiceId: "<string>", // Optional field to keep the invoice Id
  totalEarned: 123, // The total amount of the sale
  commissionRate: <number>, // Optional field to override the affiliate commission rate for this sale
}

const options = {
  method: 'POST',
  headers: {
    Authorization: 'Bearer <YOUR_API_KEY>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify(body)
};

fetch('https://www.pushlapgrowth.com/api/v1/sales', options)
  .then(response => response.json())
  .then(response => console.log(response))
  .catch(err => console.log(err));
